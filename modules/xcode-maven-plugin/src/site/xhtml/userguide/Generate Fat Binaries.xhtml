This page gives you an overview how to enable fat binaries generation for XCode Projects for Libraries.

Fat Binaries

To enable the fat binary generation (combined iphoneos and iphonesimulator binaries) for your XCode Library Project, you just have to specify the following into the pom.xml:

pom.xml 
.....  
  <build>
     <plugins>
       <plugin>
        <groupId>com.sap.prd.mobile.ios.mios</groupId>
        <artifactId>xcode-maven-plugin</artifactId>
        <executions>
          <execution>
            <id>fat-lib</id>
            <goals>
              <goal>package-fat-lib</goal>
            </goals>
            <phase>package</phase>
            <inherited>false</inherited>
            <configuration />
          </execution>
        </executions>
      </plugin>
      ...
    </plugins>
  </build>
