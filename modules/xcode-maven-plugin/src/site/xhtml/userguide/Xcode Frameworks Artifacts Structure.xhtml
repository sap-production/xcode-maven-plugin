Introduction

This page describes all the artifacts, produced out of a build of Xcode framework project with the xcode-maven-plugin.

Mandatory Artifacts

Pom File

<artifact-id>-<version>.pom

The project pom file as defined in the project. More information in Project Pom.

Versions Information

<artifact-id>-<version>-versions.xml

The file containing the reproducibility information of the project built and its (transitive) dependencies. More information in Version information.

Main Artifact

<artifact-id>-<version>.xcode-framework-zip

The framework containg the headers, the library and bundles in the appropriate format so that it can be referenced from an xcode project. The framework available as main artifact corresponds to the configuration specified by parameter "xcode.primaryFmwkConfiguration". If this parameter is not provided it defaults to "Release"

Additional Artifacts

Configuration specific frameworks

<artifact-id>-<version>$lt;classifier>.xcode-framework-zip

The classifier denotes the configuration for that the framework was built.

If a configuration specific framework version is found the configuration specific version will be used by a build for the relevant configuration. If there is no configuration specific framework is found, the default framework available as main artifact will be used instead.
